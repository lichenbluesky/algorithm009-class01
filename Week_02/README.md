##### 分治

分治算法的核心思想是将原问题划分为n个规模较小而结构与原问题相似的子问题；递归的解决这些子问题，然后再合并其结果，就得到原问题的解。
分治法分为三个阶段：

分解：将原问题分解为一系列子问题，如果子问题的规模任不够小则再继续划分，如此递归下去，直到子问题规模足够小很容易求出解为止。
解决：求解规模足够小的子问题。
合并：将求出的小规模的问题合并为一个更大规模问题的解，自底向上逐步求出原问题的解。
